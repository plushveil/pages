Introduction to Pages

Pages is a static site generator that utilizes Template Literals to create HTML files. It is designed to be simple and easy to use, while still providing a powerful tool for creating websites. Pages is built on top of Node.js and uses its native templating capabilities to generate static HTML files. Perfect for creating small to medium-sized websites, blogs, and portfolio sites.

Development server

A built-in development server and live reload feature allow you to preview your site as you work on it. npx @plushveil/pages serve . will start the development server for the current directory.

Placeholder Substitution

Define a single <link rel="canonical" href="/\${languages}/index.html"> tag to generate multiple pages, which is useful for creating SEO-friendly pages in different contexts, such as multi-language or multi-region sites. Or use script tags to define variables during the build process.

Built in minification

Out of the box Pages minifies your HTML, CSS, and JavaScript files, reducing the size of your website and improving load times.

Tailwind CSS support

Pages comes with built-in support for Tailwind CSS, a utility-first CSS framework that makes it easy to create custom designs without writing custom CSS.

GitHub Action

A GitHub Action that allows you to automate the deployment process, making it possible to only commit your website files to your repository and have the Pages GitHub Action take care of the rest. No need to store a development environment in your repository.
